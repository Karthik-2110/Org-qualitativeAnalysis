Abstract

This tool assists students in self-evaluation in practical as well as the learning of items through a user-interactive application. After completion, the score will be displayed, each observation will be given one mark, and each confirmatory test will be animated. For the sake of reference, documentation was given.

Note

This is a representation of educational software.

This application is based on the cheminfomatics idea, which provides complete access to all elements in the periodic table as well as their attributes.

It included a salt analysis test that allows students to assess themselves.

Starting the practical by pressing the start practical button takes you to the next page, where you must choose one of the cards to begin the practical.

Each question is worth 5 points, and questions that are answered incorrectly will be penalised.
